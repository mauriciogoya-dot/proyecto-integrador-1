Bienvenido a mi registro de aprendizaje

Este sitio ha sido desarrollado para la materia Proyecto Integrador 1.

No es solo un repositorio de archivos, sino un entorno de documentación activa donde registro la evolución de mis habilidades técnicas.

A través de la cursada en el ISSRC, este espacio se convertirá en mi base de conocimientos centralizada, permitiéndome repasar conceptos fundamentales de arquitectura web, protocolos de comunicación y buenas prácticas de desarrollo que aplicaré en mi carrera como Desarrollador Web."

¿Qué encontrarás aquí?

  • Guía de Terminal: El primer contacto con el sistema. Aquí documento cómo gestionar archivos y procesos sin depender de una interfaz gráfica, una habilidad crítica para cualquier desarrollador y comandos básicos para moverte en el sistema.
  • Control de Versiones: Mi flujo de trabajo con Git y GitHub.La bitácora de mis proyectos. Registro el uso de Git para el control local y GitHub para la colaboración y el despliegue en la nube.
  • Fundamentos Web: Conceptos de HTML y la estructura de este sitio. El esqueleto de la red. Notas detalladas sobre semántica HTML5, asegurando que mis sitios sean accesibles y bien interpretados por los navegadores.
  • Herramientas: Un inventario del software que potencia mi productividad, desde editores de código hasta asistentes de IA y extensiones esenciales y el software que utilizo para programar.
