3. HTML (HyperText Markup Language)

Es el Lenguaje de Marcado de Hipertexto, es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos. Define la estructura básica (el "esqueleto") mediante etiquetas para organizar párrafos, títulos, imágenes y enlaces, funcionando en todos los navegadores.

Etiquetas básicas:

  • <h1> a <h6>: Títulos (siendo h1 el más importante).
  • <p>: Párrafos de texto.
  • <ul> y <li>: Listas desordenadas (viñetas).
  • <code>: Para mostrar fragmentos de código o comandos.
