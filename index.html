Mi registro de aprendizaje

Mi guía técnica para la Tecnicatura en Desarrollo Web.

1. La Terminal (CMD / PowerShell)

Es la interfaz de texto para interactuar con el sistema operativo.

  • cd [ruta]: Cambiar de directorio (ir a una carpeta).
  • mkdir [nombre]: Crear una nueva carpeta.
  • dir: Listar los archivos de la carpeta actual.

2. Git - Control de Versiones

Herramienta para seguir los cambios en nuestro código.

  • git init: Inicia un repositorio local en la carpeta.
  • git status: Nos dice qué archivos cambiaron o no están guardados.

3. HTML (HyperText Markup Language)

Es el lenguaje que define la estructura de las páginas web mediante etiquetas.

Etiquetas básicas:

  • <h1> a <h6>: Títulos (siendo h1 el más importante).
  • <p>: Párrafos de texto.
  • <ul> y <li>: Listas desordenadas (viñetas).
  • <code>: Para mostrar fragmentos de código o comandos.
